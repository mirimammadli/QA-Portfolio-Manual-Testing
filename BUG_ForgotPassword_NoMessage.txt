Bug ID: BUG_7777
Title: Forgot Password – No confirmation message after submitting valid email
Environment: Chrome, iPhone 15 Pro Max, iOS 17
Steps to Reproduce:
1. Navigate to the login page
2. Click the “Forgot Password” link
3. Enter a valid email address
4. Click the “Submit” button
Expected Result: A confirmation message like “Reset link has been sent” should be shown
Actual Result: No confirmation message is displayed
Severity: Critical
